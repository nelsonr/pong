# Yet another Pong

This is my attempt of making a complete pong game with a hockey theme in Love2d (Lua).

### Game controls

**Player 1**: «W» and «S»

**Player 2**: «Up Arrow» and «Down Arrow»

**Pause game**: «P»

**Exit current game**: «Escape»
